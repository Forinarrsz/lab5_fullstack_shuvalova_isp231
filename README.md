## Теги в HTML

Структура парного тега

<tag>body</tag>

***Пример***
```html
<h1>Текст заголовка</h1>
```
----
## Базовые HTML-теги
#### Примеры:
```html
<h2>Заголовок</h2>
<p>Абзац текста</p>
<strong>Жирный</strong>
<em>Курсив</em>
<hr>
<a href="https://example.com">Ссылка</a>
<img src="example.jpg" alt="Описание">
```
## Основные теги и элементы
- `<input>` — поле ввода
- `<textarea>` — многострочный ввод
- `<select>` — выпадающий список
- `<button>` — кнопка
- `<label>` — подпись к элементу

*Пример:*
```html
<label for="name">Имя:</label>
<input type="text" id="name" name="name">
```


## Типичные проблемы и их решения
- *Картинка не отображается:* проверьте путь к файлу и расширение.
- *Страница отображается как код:* убедитесь, что файл сохранён с расширением `.html`.
- *Кириллица иероглифами:* добавьте `<meta charset="UTF-8">` в `<head>`.
